## 9. 既知の挙動 / 非対象（不具合ではない）

- gateway は無料枠のため**初回アクセスのみ低速**（自動起動）。常時稼働にしたい場合は Starter 化。
- プレビューURLは Vercel 認証保護（チーム限定）。一般公開は本番昇格が必要。
- 削減時間はタスク種別の**推定プリセット**（実測ではない）。係数は `routes/dashboard.ts` で調整可能。
- LLMの出力内容は都度変動（生成AIのため）。
